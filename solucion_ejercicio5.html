Cálculo de Fibonacci

A continuación se muestra la solución al ejercicio de cálculo de la secuencia de Fibonacci:

def fibonacci(n):
    if n <= 0:
        return []
    elif n == 1:
        return [0]
    elif n == 2:
        return [0, 1]
    else:
        fib = [0, 1]
        while len(fib) < n:
            fib.append(fib[-1] + fib[-2])
        return fib

numero = int(input("Ingrese un número entero positivo: "))
print("Secuencia de Fibonacci hasta", numero, ":")
print(fibonacci(numero))
