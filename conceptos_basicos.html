Variables

En Python, una variable es un contenedor para almacenar datos. Se puede pensar en una variable como un etiqueta asociada a un valor en la memoria.

Ejemplo:


# Declaración de variables
nombre = "Juan"
edad = 25

# Uso de variables
print("Hola,", nombre)
print("Tienes", edad, "años")

Tipos de Variables

Tipo de Variable Ejemplo Descripción
Entero (int) edad = 25 Se utiliza para representar números enteros.
Flotante (float) altura = 1.75 Se utiliza para representar números decimales.
Cadena (str) nombre = "Juan" Se utiliza para representar texto.
Booleano (bool) es_mayor_de_edad = True Se utiliza para representar valores de verdad.
Lista (list) numeros = [1, 2, 3, 4, 5] Se utiliza para almacenar una colección ordenada de elementos.
Tupla (tuple) colores = ("rojo", "verde", "azul") Es similar a una lista pero es inmutable.
Diccionario (dict) persona = {"nombre": "Juan", "edad": 25} Se utiliza para almacenar pares clave-valor.

Bucles

Los bucles se utilizan para iterar sobre una secuencia de elementos, como una lista o una cadena, y realizar una acción repetida.

En Python, los bucles más comunes son el bucle for y el bucle while.

Ejemplo:


# Bucle for
for i in range(5):
    print(i)

# Bucle while
contador = 0
while contador < 5:
    print(contador)
    contador += 1

Estamentos de Control

Los estamentos de control se utilizan para alterar el flujo de ejecución del programa.

Algunos estamentos de control en Python incluyen if, elif y else.

Ejemplo:


# Estamento if
x = 10
if x > 0:
    print("x es positivo")
elif x == 0:
    print("x es cero")
else:
    print("x es negativo")

Operadores

Los operadores en Python son símbolos o palabras clave que permiten realizar operaciones sobre datos y variables. Sirven para realizar una amplia variedad de tareas, desde operaciones aritméticas simples hasta comparaciones y asignaciones de valores.

Operadores Aritméticos en Python

Estos operadores se utilizan para realizar operaciones matemáticas básicas.

Operador Descripción Ejemplo Resultado
+ Suma 2 + 3 5
- Resta 5 - 2 3
* Multiplicación 3 * 4 12
/ División 10 / 3 3.3333...
// División entera 10 // 3 3
% Módulo 10 % 3 1
** Potencia 2 ** 3 8

Operadores Lógicos en Python

Estos operadores se utilizan para combinar expresiones lógicas y producir un resultado basado en la verdad o falsedad de esas expresiones. Estos operadores son útiles para tomar decisiones basadas en múltiples condiciones.

Operador Descripción Ejemplo Resultado
and AND lógico True and False False
or OR lógico True or False True
not NOT lógico not True False

Operadores de Comparación en Python

También conocidos como operadores relacionales, estos operadores se utilizan para comparar dos valores. Estos operadores devuelven valores booleanos (True o False) que indican si la comparación es verdadera o falsa.

Operador Descripción Ejemplo Resultado
== Igual a 5 == 5 True
!= Diferente de 5 != 5 False
< Menor que 5 < 10 True
> Mayor que 10 > 5 True
<= Menor o igual que 5 <= 5 True
>= Mayor o igual que 5 >= 5 True

Operadores de Asignación en Python

Estos operadores se utilizan para asignar valores a variables. Además del operador de asignación básico, Python también tiene operadores de asignación compuestos que realizan una operación aritmética y luego asignan el resultado a la variable.

Operador Descripción Ejemplo Resultado
= Asignación x = 5 x es igual a 5
+= Suma y asignación x += 3 x es igual a x + 3
-= Resta y asignación x -= 3 x es igual a x - 3
*= Multiplicación y asignación x *= 3 x es igual a x * 3
/= División y asignación x /= 3 x es igual a x / 3
//= División entera y asignación x //= 3 x es igual a x // 3
%= Módulo y asignación x %= 3 x es igual a x % 3
**= Potencia y asignación x **= 3 x es igual a x ** 3
