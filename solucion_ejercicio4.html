Números Primos

A continuación se muestra la solución al ejercicio de generación de números primos:

def es_primo(n):
    if n <= 1:
        return False
    for i in range(2, int(n ** 0.5) + 1):
        if n % i == 0:
            return False
    return True

limite = int(input("Ingrese un número entero positivo: "))
print("Números primos menores que", limite, ":")
for numero in range(2, limite):
    if es_primo(numero):
        print(numero, end=" ")
