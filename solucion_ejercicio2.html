Factorial

A continuación se muestra la solución al ejercicio de cálculo del factorial:

def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)

numero = int(input("Ingrese un número entero positivo: "))
print("El factorial de", numero, "es:", factorial(numero))
